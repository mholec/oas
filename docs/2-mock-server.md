# Mock Server

Pomocí mock serveru lze přijímat nebo odesílat testovací data na API. Informace o mock serveru naleznete po kliknutí na detaily [API reference](/docs/oas/83330f7496caa-senzory).

## Veřejný server
Aktuální URL mock serveru online je :

```text
https://stoplight.io/mocks/miroslavholec/oas/-1
```

## Lokální server
Chcete-li spustit mock server lokálně, můžete k tomuto účelu použít nástroj [Prism](https://stoplight.io/open-source/prism). Máte-li nástroj nainstalovaný jako CLI, pak stačí spustit v terminálu příkaz:

```shell
prism mock ./path/to/yaml-file.yaml
```

Detailní informace k mockování najdete v [dokumentaci](https://docs.stoplight.io/docs/prism/83dbbd75532cf-http-mocking).
