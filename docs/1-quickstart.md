# Quickstart Guide

Tato stránka je ukázkou, jak může vypadat dokumentace REST API. Součástí této dokumentace je i kompletní API reference se seznamem všech endpointů. 

API je zaměřeno na téma sběru informací ze senzorů. Nemá žádnou funkční implementaci, ale je možné se napojit na [mock server](/7s7hhc2n5lmi8-mock-server) vytvořený pomocí prism.
