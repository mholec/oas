# Quickstart Guide

Tato stránka je ukázkou, jak může vypadat **dokumentace REST API**. Součástí této dokumentace je i kompletní API reference se seznamem všech endpointů. 

API je zaměřeno na téma sběru informací ze senzorů. Nemá žádnou funkční implementaci, ale je možné se napojit na [mock server](/docs/oas/7s7hhc2n5lmi8-mock-server) vytvořený pomocí prism. 

Toto API používám jako ukázku na mých školeních, přednáškách a také v rámci mého Průvodce Designem REST API.

- [Školení REST API Design](https://www.miroslavholec.cz/skoleni/rest-api-design)
- [Průvodce Designem REST API](https://www.restapi.cz)
